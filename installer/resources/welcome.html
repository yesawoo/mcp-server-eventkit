MCP EventKit Server

Integra Apple Reminders con Claude Desktop usando el Model Context Protocol (MCP).

Funcionalidades:

✓ Listar y buscar recordatorios
✓ Crear nuevos recordatorios
✓ Marcar como completados
✓ Actualizar recordatorios existentes
✓ Listar tags (requiere Full Disk Access)

Requisitos:

  • macOS 13.0 (Ventura) o superior
  • Claude Desktop instalado
